# Learning Go

Interactive course app for Go syntax, runtime behavior, and concurrency patterns.

## Runtime

- Node.js 25.9.0

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`

## Lessons

- `package main` and program entry
- Slices and backing arrays
- Map lookup semantics
- Struct methods and pointer receivers
- Interfaces and implicit satisfaction
- Goroutines
- Channels
- Context cancellation

## Workspace

- Go Playground: https://go.dev/play/
